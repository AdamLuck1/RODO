# RODO-Assessment

# To setup backend

1) Navigate to the Backend directory from the root directory.
2) Use "npm install" to install node modules
3) To run the dev server use "npm run start"
4) To run tests use "npm run test"


# To setup frontend

1) Use "npm install" to install node modules
2) To run the dev server use "npm run dev"
3) To run tests use "npm run test"
